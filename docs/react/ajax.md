# ajax

## 常用ajax库

1. jQuery:比较重
2. axios:轻量级
   - 封装XmlHttpRequest对象的ajax
   - promise风格
   - 可以用在浏览器以及node服务器端
3. fetch:原生函数,但是老版本浏览器不支持
   - 为了兼容低版本浏览器,引入兼容库fetch.js
